# Resume Website

A simple resume webpage built using HTML and minimal CSS, designed to resemble a clean, traditional resume layout suitable for evaluation and PDF export.

## Features

Clean and readable resume layout

Uses only HTML + minimal CSS

ATS-friendly structure

Easy to customize

Printable as PDF

## Tech Stack

HTML5

CSS3 (minimal, no frameworks)

## Follow these steps to run the project locally:

Clone the repository

git clone https://github.com/reaperggs/chaicode-resume-assignment.git


Navigate to the project folder

cd chaicode-resume-assignment


Open the file
Simply open index.html in any browser.
