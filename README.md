# Resume Website

A simple resume webpage built using HTML and minimal CSS. The layout follows a clean and traditional resume structure suitable for evaluation and PDF export.

## Setup Instructions

Follow these steps to run the project locally.

1. Clone the repository

```bash
git clone https://github.com/reaperggs/chaicode-resume-assignment.git
```

2. Navigate to the project folder

```bash
cd chaicode-resume-assignment
```

3. Open the file

Open index.html in any web browser.
